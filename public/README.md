# Public Assets

This directory contains static assets served by Next.js.

## Required Images

Please add the following images to this directory:

1. **michael-hero.jpg** - Hero image for the homepage (recommended size: 600x800px or similar aspect ratio)
   - **Recommended:** The surfing photo (man on wave with peace sign) - embodies ocean freedom theme
   - **Alternative:** Couple with dog photo or kayaker at sunset

2. **michael-headshot.jpg** - Circular headshot for the "Meet Michael" section (recommended size: 320x320px or square)
   - **Recommended:** The bald man in red rashguard smiling at camera - friendly, professional, warm
   - **Alternative:** Cropped from family photos if that's Michael

These images are referenced in:
- `app/(public)/page.tsx` - Homepage hero and "Meet Michael" sections

See `IMAGE_GUIDE.md` for detailed recommendations based on the images you've provided.
