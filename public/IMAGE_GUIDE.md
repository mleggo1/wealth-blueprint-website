## How to Add Images

1. Save the selected images to this `/public` directory
2. Name them exactly:
   - `michael-hero.jpg`
   - `michael-headshot.jpg`
3. Ensure good image quality (high resolution)
4. The website will automatically display them once added

## Image Optimization Tips

- **Hero image:** Should be high quality, well-lit, and show Michael in a positive, active way
- **Headshot:** Should be a clear, professional portrait with good lighting
- Both images should be optimized for web (compressed but not pixelated)
- Recommended format: JPG or WebP
